オセロゲーム
<概要>
二種類のオセロAIと対戦できるオセロゲーム

<環境>
Windows

<開発>
C++
DXライブラリを使用

<起動方法>
picディレクトリ、soundディレクトリ、x64/Release/Othello.exeを同じディレクトリに配置してOthello.exeを実行

<終了方法>
Windows標準の終了ボタンではなく、タイトル画面右上のゲーム内終了ボタンを押す

<操作方法>
全てマウスの左クリック

-----------------------------------------------------------------
<タイトル画面>
・BLACK(先手)かWHITE(後手)を選択可能
・AIの種類を選択（MCTS or AlphaBeta）
・AIの強さを選択
・STARTボタンでゲーム開始

<ゲーム画面>
・マウスの左クリックで駒を配置
・左上×ボタンでタイトル画面に戻る
・右上のボタンで一手戻る
・決着すると自動的にリザルト画面へ遷移

<リザルト画面>
・NEXTボタンでタイトル画面に遷移
・RETRYボタンでゲーム画面に遷移

----------------------------------------------------------
<AI>
MCTS(モンテカルロ木探索)とαβ法の二種類

<盤面情報>
ビットボードを用いて高速化
